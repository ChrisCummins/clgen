### Requirements

* Ubuntu 16.04 or macOS.
* OpenCL.


### Installation

Clone this repository and use the [bootstrap](/tools/bootstrap.sh) script to
prepare the system dependencies and toolchain:

```sh
$ https://github.com/ChrisCummins/phd.git
$ cd phd
$ ./tools/bootstrap.sh
```

Test the universe using:

```
$ bazel test //...
```
