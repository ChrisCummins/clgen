# Plausible Test Case Generation for Compiler and Runtime Bug Discovery


## Requirements

* Ubuntu 16.04
* Python 3.6
* Nvidia GPU and CUDA

## Installation

Checkout all submodules:

```sh
$ git submodule update --init --recursive
```

Create the virtualenv and build all libs:

```
$ make all
```


## Running the code

Launch the Jupyter server:

```
$ make run
```
